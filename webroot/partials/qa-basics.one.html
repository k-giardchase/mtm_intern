JIRA

Familiarizing yourself with the bug tracking software at Metal Toad is important in understanding how a typical QA workflow operates. You will need a few things to get started:

  • Review the JIRA 101 guide (specifically the create an issue and search for issues sections)
  • Installation of JIRA Capture (Follow the Capture link located in the dashboard of our JIRA website for installation on different browsers)
  • Login credentials (request access from Tyler Ward)
  • Project specific access (request access from admin or project manager)
