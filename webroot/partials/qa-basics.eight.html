Behavior Driven Development + Behat

The user story describes a more specific action that a user may take within that epic. In order to cover all the actions a user might need to fulfill the epics theme many user stories will be nested within that epic. An example user story might be:

As a user (ROLE)
I want to be able to view my purchase history (ACTION)
So that I can find past purchases (BENEFIT)
